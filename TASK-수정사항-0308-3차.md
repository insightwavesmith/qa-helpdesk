## T1. 경쟁사 분석기 — 핀(모니터링 등록) + 더보기

### 핀
검색 결과에서 브랜드 핀 클릭 시 "브랜드명을 입력해주세요" 에러.
SearchAPI.io 응답에서 `page_name`이 null인 경우가 있다. 이때 `name`, `page_alias`, `ig_username` 등 다른 필드로 대체해야 한다.

### 더보기
"더보기" 클릭 시 "더 볼 광고가 없습니다". 30개 이후 광고가 있는데 로드 안 됨.
중복 제거 후 새 광고 0건이면 `nextPageToken`을 null로 설정하는 게 원인. 중복이어도 토큰을 유지하고 다음 페이지를 시도해야 한다.

### 파일
- `src/lib/meta-ad-library.ts` — page_name null fallback
- `src/app/(main)/protractor/competitor/` — 더보기 페이지네이션 로직

### 하지 말 것
- 검색 드롭다운 건드리지 마라
- 필터/정렬 건드리지 마라

## T3. 개인정보처리방침 보관기간 수정

현재 서비스 이용기록 보관기간이 3개월로 되어있다. 법적으로 장기보관 가능하고, 서비스 운영에 필요하다.

### 파일
- `src/app/privacy/page.tsx`

### 수정 내용
- 서비스 이용기록: 3개월 → **회원 탈퇴 시까지** (또는 서비스 종료 시까지)
- 광고 데이터 / 자사몰 데이터: **계약 종료 후 5년** (전자상거래법)

  - 자사몰 데이터 (메타 광고 계정 연동을 통한 광고 성과 데이터)
  - 광고 데이터 (일일 광고 인사이트, 타겟 중복 분석 데이터)
  - 서비스 이용 분석 (Mixpanel을 통한 사용 패턴 분석)
- 보관기간별 분리 표시
