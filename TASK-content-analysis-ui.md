# TASK: 소재 분석 탭 UI/UX 목업 구현

## ⚠️ 최우선 원칙
**기존 bscamp UI/UX를 먼저 정확히 파악해라.** 기존 서비스를 깨뜨리지 않고, 기존 구조·네비·스타일을 100% 유지하면서 소재 분석 기능을 "추가"하는 방향이다. 새로 뜯어고치는 게 아니라, 기존 흐름에 자연스럽게 녹이는 것.

### 필수 사전 작업
1. 기존 bscamp 라우팅 구조 파악 (app/ 폴더 전체)
2. 기존 사이드바 메뉴 구조 파악
3. 기존 탭/카드/테이블 컴포넌트 스타일 파악
4. 기존 페이지들이 어떤 레이아웃 패턴 쓰는지 파악
5. 위 파악 후 → 소재 분석 탭을 "어디에, 어떻게" 넣을지 설계
6. 기존 컴포넌트 재사용 최대화 — 새 컴포넌트는 최소화

## 고객 관점
수강생이 자기 광고 소재(이미지/영상/LP)의 AI 분석 결과를 보는 화면이 필요하다.
현재 bscamp 대시보드에 "소재 분석" 탭을 추가해서, 소재별 4축 분석 결과 + 시선 히트맵 + 성과 매핑을 한 화면에서 볼 수 있어야 한다.

## 기대 동작

### 1. 소재 분석 대시보드 (메인 뷰)
- 상단: 전체 소재 통계 카드 (이미지 2,835 / 영상 261 / LP ~35 / 분석 완료율)
- 필터: 소재 유형(이미지/영상/LP) / 수강생 / 기간 / 성과순 정렬
- 소재 그리드: 썸네일 + 주요 속성 태그 + ROAS 표시
- 기존 bscamp UI/UX 스타일 100% 동일 (다크 테마, shadcn/ui, 사이드바 네비)

### 2. 소재 상세 분석 (개별 소재 클릭 시)
- **이미지**: 원본 + 시선 히트맵 오버레이 토글
- **영상**: 원본 영상 + Canvas 시선 히트맵 오버레이 (초별 이동)
  - 영상 재생되면서 히트맵 원이 시선 따라 움직임
  - 구간별 색상: 0-3초 훅(빨강) / 3-8초 제품(파랑) / 8-15초 CTA(초록)
  - 재생/일시정지/탐색 가능
- 4축 분석 결과: Visual/Audio/Text/Structure 탭
- 성과 데이터: ROAS, CTR, CPC, CPM 카드

### 3. 속성값 교차분석 뷰
- 속성값별 평균 ROAS 차트 (hook_type별, visual_style별 등)
- 고성과 패턴 하이라이트
- 속성 조합 필터 (예: hook_type=문제제기 + emotion=불안 → 평균 ROAS)

### 4. 영상 히트맵 플레이어 (핵심 신규 기능)
- 원본 영상 위에 Canvas 레이어
- 초별 fixation 좌표를 가우시안 블러 원으로 렌더링
- 타임라인 바: 0-3초 훅 / 3-8초 제품 / 8-15초 CTA 구간 표시
- 재생 컨트롤바

### 5. LP 분석 뷰
- LP 목록 (전환 점수 0~100 표시)
- 개별 LP: 모바일/PC 스크린샷 + 8개 카테고리 분석 결과
- 소재↔LP 일관성 점수
