## T1. 개인정보처리방침 — 광고 데이터 수집 내용 추가

개인정보처리방침에 자사몰/광고 데이터 수집 내용이 빠져있다. 수강생의 자사몰 데이터와 광고 데이터를 수집하고 있으므로 반드시 명시해야 한다.

### 파일
- `src/app/privacy/page.tsx`

### 현재 동작
- "이용 데이터" 항목에 검색 기록, 콘텐츠 열람, 광고 분석 기록만 있음

### 기대 동작
- "이용 데이터" 항목에 다음 추가:
  - 자사몰 데이터 (메타 광고 계정 연동을 통한 광고 성과 데이터)
  - 광고 데이터 수집 (일일 광고 인사이트, 타겟 중복 분석 데이터)

### 하지 말 것
- 다른 섹션 건드리지 마라

## T2. 경쟁사 분석기 "더보기" 여전히 안 됨

1차 수정에서 더보기 수정했다고 했는데 여전히 동작 안 한다. 왜 안 되는지 원인부터 파악하고 수정해라.

### 파일
- `src/app/(main)/protractor/competitor/` 관련 파일

### 현재 동작
- "더보기" 클릭 → 추가 광고 로드 안 됨
- 드롭다운도 다시 뜨지 않음

### 기대 동작
- "더보기" 클릭 → 다음 페이지 광고 카드가 기존 목록 아래에 추가
- 전체 건수만큼 반복 로드 가능

### 하지 말 것
- 검색 기능 건드리지 마라
- 필터/정렬 건드리지 마라

## T3. 경쟁사 분석기 핀(모니터링 등록) 여전히 안 됨

1차 수정에서 핀 수정했다고 했는데 여전히 동작 안 한다. 왜 안 되는지 원인부터 파악하고 수정해라.

### 파일
- `src/app/(main)/protractor/competitor/` 관련 파일

### 현재 동작
- 검색 결과 브랜드 옆 핀 버튼 클릭 → 모니터링에 등록 안 됨

### 기대 동작
- 핀 클릭 → 모니터링 패널에 브랜드 추가
- 등록 성공 시 토스트 알림
- 이미 등록된 브랜드는 핀 비활성화

### 하지 말 것
- 모니터링 패널의 기존 기능 건드리지 마라
- 검색 드롭다운 건드리지 마라

## 중요
T2, T3는 1차에서 수정 완료라고 했는데 실제로 안 됐다. **왜 이전 수정이 동작하지 않는지 원인을 먼저 파악**하고 수정해라. 원인 모르고 수정하면 또 같은 일 반복된다.

## 검증 기준
- /privacy 페이지에 자사몰/광고 데이터 수집 내용 표시
- 더보기 클릭 → 추가 광고 카드 로드
- 핀 클릭 → 모니터링 패널에 등록
- npm run build 성공
- 커밋+푸시까지
