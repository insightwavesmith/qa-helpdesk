## T1. [HIGH] 도달당구매율 미표시

### 현상
성과요약 탭 전환 지표에 4개만 표시 (CTR, 결제시작율, 구매전환율, 결제→구매율).
5번째 "도달당구매율"(reach_to_purchase_rate) 없음.

### 원인
T3_PARTS 또는 UI 컴포넌트에서 reach_to_purchase_rate가 전환 그룹에 포함되지 않았거나, 표시 로직에서 빠졌을 가능성.

### 수정 방향
1. `src/lib/protractor/t3-engine.ts` — T3_PARTS conversion 그룹에 reach_to_purchase_rate 포함 확인
2. 성과요약 UI 컴포넌트에서 전환 지표 5개 전부 표시되도록 수정
3. 도달당구매율 라벨: "도달당구매율" (= purchases / impressions × 100)
4. computeMetricValues에서 reach_to_purchase_rate 리턴 확인

## T2. [MEDIUM] 콘텐츠 카드 CTR 위치 불일치

### 현상
- 1등 카드: CTR이 "전환율" 섹션에 위치
- 2~5등 카드: CTR이 "기반점수" 섹션에 위치
- 총 13개 지표 수는 동일하나 섹션 배치가 카드마다 다름

### 원인
1등 카드와 2~5등 카드의 렌더링 컴포넌트 또는 지표 그룹핑 로직이 다른 듯.

### 수정 방향
모든 카드에서 CTR을 동일한 섹션(전환율)에 배치.
지표 그룹핑:
- 영상(3): 3초재생률, 끝까지시청률, 이탈률
- 참여(5): 좋아요, 댓글, 공유, 저장, 참여합계 /만노출
- 전환(5): **CTR**, 결제시작율, 구매전환율, 결제→구매율, 도달당구매율

### 검증
콘텐츠 탭에서 1~5등 카드 전부 동일한 섹션 구조 확인
