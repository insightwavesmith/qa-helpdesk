# TASK: 수강생 랭킹 기능

## 무엇을 (What)
수강생별 + 팀별 랭킹 페이지를 추가한다. 관리자와 수강생 모두 열람 가능.

## 왜 (Why)
- 3주차부터 수강생 간 선의의 경쟁 유도
- 총가치각도기 기반 성과를 **성장률**로 비교 (절대값이 아닌 성장률 → 공평)
- 팀별 랭킹으로 팀 내 협동 유도

## 기능 요구사항

### 수강생 랭킹
- 기준: 총가치각도기(T3) 성장률
- 성장률 = (현재 총가치 - 3주차 베이스라인) / 베이스라인 × 100%
- 분모 0 문제: 최초 수집값을 베이스라인으로 사용
- 표시: 순위, 이름(or 브랜드), 현재 점수, 성장률, 등급(A~F)
- 정렬: 성장률 DESC

### 팀별 랭킹
- 팀원 성장률 평균으로 팀 순위
- 팀 매핑: DB에 team 컬럼 필요 (profiles 테이블)
- 팀 구성은 Smith님이 직접 배정 (관리자 회원관리에서)

### 접근 권한
- 관리자: 전체 열람
- 수강생: 전체 열람 (동기부여 목적)
- 미승인/리드: 접근 불가

## 관련 파일
- `src/app/(main)/protractor/` (총가치각도기)
- `t3_scores_precomputed` 테이블 (사전계산 데이터)
- `student_performance_daily` 테이블 (수강생 성과)
- `src/types/database.ts` (profiles 타입)

## 검증 기준
1. 수강생 랭킹 페이지 정상 렌더링
2. 팀별 랭킹 정상 렌더링
3. 성장률 계산 정확성 (베이스라인 대비)
4. 관리자/수강생 접근 가능, 리드 접근 차단
5. `tsc --noEmit` 에러 없음
6. `next build` 성공
