# TASK: Q&A 글 수정 기능 추가

---

## 빌드/테스트
- `npm run build` 성공 필수

---

## 이게 뭔지
수강생이 Q&A에 질문을 올린 후 수정할 수 없음. 오타, 이미지 교체, 내용 보충이 불가능.

## 왜 필요한지
수강생이 질문을 올렸는데 이미지가 잘못되거나 글에 오류가 있으면 삭제 후 재작성해야 함. 수정 기능이 있으면 기존 글에서 바로 고칠 수 있음.

## 기대 동작
1. Q&A 상세 페이지(`/questions/[id]`)에서 **본인이 작성한 글**에 "수정" 버튼 표시
2. 수정 버튼 클릭 → 기존 글 작성 UI/UX 그대로 본문 영역이 편집 모드로 전환
3. 제목, 본문(마크다운), 이미지, 카테고리 수정 가능
4. "저장" 버튼으로 수정 완료 → 기존 글이 업데이트됨
5. 본인 글만 수정 가능 (다른 사람 글에는 수정 버튼 없음)
6. 관리자(admin/assistant)는 모든 글 수정 가능

## 기존 글 작성 UI 참고
- `/questions/new` 페이지의 글 작성 폼 — 이 UI/UX를 수정 모드에서 그대로 재활용
- 제목 입력, 카테고리 선택, 본문 에디터, 이미지 첨부

## 파일 (참고용, 직접 탐색해서 확인)
- `src/app/(main)/questions/[id]/page.tsx` — 상세 페이지
- `src/app/(main)/questions/new/` — 글 작성 페이지 (UI 재활용 대상)
- `src/actions/questions.ts` — Server Actions
