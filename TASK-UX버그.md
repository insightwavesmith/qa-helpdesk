## 개요

관리자 수강생 관리 + 수강생 온보딩/설정 화면의 UX 버그 4건 수정.

---

## T1. 관리자 수강생 탭 — 믹스패널ID/광고계정 수정 가능하게

- 현재: 관리자가 수강생 상세에서 믹스패널 프로젝트 ID, 광고계정 ID를 볼 수만 있고 수정 불가
- 변경: 인라인 편집 또는 수정 모달로 변경 가능하게
- 대상 파일: 수강생 상세/편집 관련 컴포넌트 (관리자 페이지)
- 수정 후 Supabase `ad_accounts` 또는 `profiles` 테이블 업데이트 연동

## T2. 수강생 온보딩 입력창 — 로그인으로 돌아가기 버튼 추가

- 현재: 수강생이 온보딩(믹스패널/광고계정 입력) 화면에서 로그인 화면으로 돌아가는 방법 없음
- 변경: "로그인으로 돌아가기" 또는 "뒤로가기" 버튼 추가
- 대상 파일: 온보딩 페이지 컴포넌트
- 동작: 클릭 시 로그아웃 + 로그인 페이지로 이동 (또는 단순 뒤로가기)

## T3. 수강생 설정창 — 믹스패널/광고계정 재입력 가능

- 현재: 수강생이 온보딩 때 입력한 믹스패널ID/광고계정을 이후 수정할 수 없음
- 변경: 수강생 설정(마이페이지/프로필) 화면에서 믹스패널 프로젝트 ID, 광고계정 ID 수정 가능
- 설정 화면이 없으면 신규 생성 (사이드바 메뉴에 "설정" 추가)
- 대상 파일: 수강생 대시보드 사이드바 + 설정 페이지

## T4. 믹스패널 대시보드 ID 입력 기능 추가

- 현재: 수강생/관리자가 믹스패널 프로젝트 ID만 입력 가능, 대시보드 ID 입력란 없음
- 필요 이유: 총가치각도기 TOP 5 광고의 "믹스패널" 버튼 클릭 시 해당 대시보드로 필터 걸어서 이동해야 함
- 변경:
  - 온보딩 화면: 믹스패널 대시보드 ID 입력 필드 추가
  - 수강생 설정창: 믹스패널 대시보드 ID 수정 가능
  - 관리자 수강생 탭: 믹스패널 대시보드 ID 수정 가능
  - DB: `profiles` 또는 `ad_accounts` 테이블에 `mixpanel_board_id` 컬럼 추가 (없으면)
  - 총가치각도기 믹스패널 버튼: `https://mixpanel.com/project/{project_id}/view/{board_id}` 형태로 링크 생성

## T5. 초대코드 만료일 기능 동작 안 함

- 현재: 관리자가 초대코드 생성 시 만료일을 설정할 수 있지만, 실제로 만료일이 지나도 코드가 사용 가능 (만료 체크 안 됨)
- 변경: 초대코드 사용 시 만료일 검증 로직 추가
- 대상: 초대코드 검증 API (회원가입 또는 코드 입력 시점)
- 만료된 코드 사용 시 "만료된 초대코드입니다" 에러 메시지 표시
