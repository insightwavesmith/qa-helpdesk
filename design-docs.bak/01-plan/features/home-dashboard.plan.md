# 홈 대시보드 계획서

## 1. 개요
- 기능 설명: 사용자 역할별 맞춤형 홈 화면과 대시보드 제공으로, 핵심 정보와 기능에 빠르게 접근할 수 있는 중앙 허브 역할
- 해결하려는 문제:
  - 사용자별 중요한 정보의 일목요연한 제공 부재
  - 새로운 질문, 답변, 공지사항에 대한 신속한 인지 어려움
  - 관리자의 서비스 운영 현황 파악 도구 필요

## 2. 핵심 요구사항

### 기능적 요구사항
- FR-01: 사용자 역할(학생/관리자)에 따라 다른 대시보드가 표시되어야 한다
- FR-02: 학생 대시보드에는 최신 공지사항, 인기 질문, 최근 질문을 표시해야 한다
- FR-03: 관리자 대시보드에는 서비스 통계, 승인 대기 현황, 주간 질문 추이를 표시해야 한다
- FR-04: 주요 통계는 실시간으로 업데이트되어야 한다
- FR-05: 대시보드에서 주요 기능으로 바로가기 링크를 제공해야 한다
- FR-06: 최근 활동 내역을 시간순으로 표시해야 한다
- FR-07: 그래프와 차트를 통해 데이터를 시각적으로 표현해야 한다

### 비기능적 요구사항
- 성능: 대시보드 로딩 시간 < 2초
- 사용성: 모바일 화면에서도 최적화된 레이아웃 제공
- 접근성: 색상 대비와 폰트 크기 접근성 기준 준수

## 3. 용어 정의

| 용어 | 영문 | 설명 |
|------|------|------|
| 대시보드 | Dashboard | 주요 정보와 기능을 한눈에 볼 수 있는 메인 화면 |
| 위젯 | Widget | 대시보드 내 개별 정보 표시 단위 |
| 통계 카드 | Stats Card | 숫자 형태의 주요 지표를 표시하는 컴포넌트 |
| 활동 피드 | Activity Feed | 최근 발생한 활동들의 시간순 목록 |
| 바로가기 | Quick Action | 주요 기능으로 즉시 이동할 수 있는 링크 |

## 4. 범위

### 포함
- 역할별 맞춤형 대시보드 (학생/관리자)
- 실시간 통계 표시
- 최근 질문/답변/공지사항 요약
- 승인 대기 항목 현황 (관리자용)
- 주간 질문 추이 그래프
- 주요 기능 바로가기 메뉴
- 반응형 레이아웃

### 제외
- 대시보드 레이아웃 커스터마이징
- 개인별 위젯 설정
- 상세 분석 리포트
- 데이터 내보내기 기능
- 실시간 알림 센터

## 5. 성공 기준

- [ ] 학생 사용자에게 학생용 대시보드가 표시된다
- [ ] 관리자 사용자에게 관리자용 대시보드가 표시된다
- [ ] 전체 질문 수, 금주 질문 수, 미해결 질문 수가 표시된다
- [ ] 승인 대기 답변 수가 실시간으로 표시된다 (관리자용)
- [ ] 최근 4주간 일별 질문 수 그래프가 표시된다 (관리자용)
- [ ] 최근 질문 5개가 제목과 함께 표시된다
- [ ] 최근 공지사항 5개가 표시된다
- [ ] 질문하기, 공지사항 등 주요 기능으로 바로가기가 가능하다
- [ ] 모바일 화면에서도 적절하게 레이아웃이 조정된다
- [ ] 대시보드 데이터가 페이지 로드 시 자동으로 업데이트된다
