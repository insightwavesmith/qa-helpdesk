Pixel과 Conversions API 완전정복
"전환 추적이 안 되면, 메타 AI는 눈을 감고 광고하는 거예요."
광고비 100만 원을 쓰는데 전환이 3건밖에 안 잡힌다면?
실제론 10건인데 7건이 누적되고 있을 수 있어요. Pixel(픽셀)과 CAPI(전환 API) 세팅이 빠져있기 때문이에요.
INSIGHT
Pixel(픽셀)은 고객의 행동을 추적하는 눈이에요. 웹사이트 방문, 장바구니 담기, 구매까지 모든 행동을 메타에 알려주는 코드예요.

문제는 iOS 14.5 이후 브라우저 추적이 30~40% 막힌다는 거예요. CAPI(전환 API)는 이 문제의 해결책이에요.
💡 실제 사례: Pixel만 쓰던 쇼핑몰이 CAPI를 추가 연동한 후, 잡히는 전환 수가 42% 증가했어요. 광고비는 그대로인데 ROAS가 1.8 → 3.1로 올랐습니다.
KEY POINT
01
Pixel 베이스 코드 = 모든 페이지에 설치
메인 페이지만 넣으면 안 돼요. 상품 상세, 장바구니, 결제 완료 페이지 전부에 들어가야 해요. 이벤트별 코드도 따로 설정해야 해요.
02
CAPI 이중 추적 = 2026년 기본 세팅
Shopify는 앱 하나로, 카페24는 플러그인으로 연동할 수 있어요. 설정 난이도가 높지 않아요.
03
EMQ(이벤트 매칭 품질) 8점 이상이 목표
이메일, 전화번호 같은 고객 정보를 전환 이벤트에 포함하면 EMQ 점수가 올라가요. 8점 이상이면 메타 AI 최적화가 제대로 작동합니다.
CHECKLIST
✓ Pixel 베이스 코드가 전체 페이지에 설치되어 있나요?
✓ 구매·장바구니·결제시작 이벤트가 정상 발화하나요?
✓ CAPI가 연동되어 서버 사이드 추적이 작동하나요?
✓ EMQ 점수가 8 이상인가요?
✓ 이벤트 중복 제거(Deduplication)가 설정되어 있나요?
하나라도 빠져있다면, 지금 광고비의 30%가 허공에 사라지고 있는 거예요.
전체 세팅 가이드에서 단계별로 확인해보세요.

Template B: 웨비나 (수정 반영 — hookLine 옵션 B)
LIVE 무료 웨비나

메타 광고는 확률 게임이다 — 총가치 기반 광고·랜딩·구매 여정 완전 해부 웨비나

왜 같은 예산을 써도 누구는 ROAS 800%이고, 누구는 200%일까요?
그 답은 메타가 광고를 노출시키는 경매 구조에 숨어 있어요. 단순히 돈을 많이 쓴다고 이기는 경매가 아니라, 총가치(Total Value)라는 점수판 위에서 벌어지는 확률 게임이에요. 수백 개 자사몰의 메타 광고를 직접 운영·최적화해 온 자사몰사관학교가 그 구조를 라이브로 공개해요.
강의 미리보기
▶ 강의 슬라이드 미리보기
밑줄 친 이미지를 교체해주세요
메타 경매 구조 · 총가치각도기 · 믹스패널 데이터 분석 슬라이드
핵심 주제
01
메타 경매의 진짜 구조 — 총가치(Total Value) 완전 해부
입찰가 × 예상 행동률 × 광고 품질, 이 세 가지를 곱한 총가치 점수로 경매 승패가 갈려요.
02
총가치각도기로 내 광고 위치 시각 진단하기
광고 소재 품질, 랜딩페이지 체류 시간, 구매 전환율 — 이 세 축의 기울기를 각도처럼 한눈에 파악하고 약한 지점을 즉시 찾아내요.
03
믹스패널(Mixpanel) 실시간 라이브 시연
GA4만으로는 부족한 이유를 보여드리고, 유저가 자사몰에서 어디서 멈추고 어떤 행동 후 구매하는지 이벤트 단위로 추적하는 법을 실제 대시보드로 시연해요.
이런 분들을 위해
• 자사몰 매출이 정체되어 돌파구가 필요하신 대표님
• 광고비는 쓰는데 수익이 남는 건지 확신이 없는 분
• 메타 광고를 직접 성장하지만 체계적인 방법론이 없는 분
• 다음 기수 수강을 고민 중이라 커리큘럼을 먼저 체험하고 싶은 분

핵심 변화
01
타겟 전략: 감에서 데이터로

02

"이번 소재 느낌이 좋아요" → "이 소재 CTR이 2.8%니까 예산 배분 늘려요"로 의사결정 기준이 바뀌었어요.
03
데이터 추적: CAPI 완전 연동
Pixel만 있을 때 전환 데이터 30% 누락 → CAPI 이중 추적으로 메타 AI 최적화 극대화. EMQ 9점 달성.
