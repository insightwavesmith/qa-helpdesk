# AI 답변 수정 시 이미지 추가 기능 — Plan

## 요구사항
관리자가 `/admin/answers` 페이지에서 AI 답변 수정 시 이미지도 추가/삭제할 수 있어야 한다.

## 배경
- AI 답변에 참고 이미지(스크린샷, 예시)를 붙여 품질 향상
- 수강생 질문에 시각적 답변이 필요한 경우 대응

## 범위
- 수정 모드에 이미지 업로드 UI 추가 (Textarea 아래)
- 기존 이미지 표시 + 삭제(X) 버튼
- 새 이미지 업로드 → Supabase Storage 직접 업로드
- `updateAnswer()` 서버 액션에 `imageUrls` 파라미터 추가

## 범위 외
- 새 API 라우트 생성 금지
- answers 테이블 스키마 변경 금지 (image_urls 컬럼 이미 존재)
- 이미지 리사이징/압축 불필요

## 성공 기준
- 답변 수정 모드에서 이미지 추가 가능
- 미리보기 + 삭제(X) 동작
- 저장 후 이미지 표시
- 기존 텍스트 수정 기능 정상
- `tsc --noEmit` 통과
