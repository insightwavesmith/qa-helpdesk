# 전화번호 필수화 + 관리자 전화번호 편집

## 배경
- 답변 완료 시 카카오 알림톡 발송에 전화번호 필요
- 현재 수강생 대부분 전화번호 미입력 상태
- 기존 가입자는 관리자가 직접 입력해야 함

## 범위
1. 수강생 가입 시 전화번호 필수 입력으로 변경
2. 관리자 회원관리 페이지에서 전화번호 컬럼 표시 + 인라인 편집

## 수정 파일
- `src/app/(auth)/signup/page.tsx` — 가입 폼 validation + UI
- `src/app/(main)/admin/members/members-client.tsx` — 회원 목록 전화번호 컬럼 + 인라인 편집

## 하지 말 것
- profiles 테이블 스키마 변경 금지 (phone 컬럼 이미 존재)
- 기존 가입자 데이터 자동 마이그레이션 금지
- 수강생 마이페이지 전화번호 수정 기능은 스코프 밖

## 성공 기준
- 수강생 모드 가입 시 전화번호 미입력 → 가입 불가
- 전화번호 형식 오류 시 에러 메시지 표시
- 관리자 회원 목록에 전화번호 표시
- 관리자가 인라인으로 전화번호 입력/수정 가능
- tsc + lint + build 통과
