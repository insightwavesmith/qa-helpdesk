## 1. 개요

자사몰사관학교(bscamp) 오가닉 마케팅 5대 채널의 콘텐츠 유형, 자동화 파이프라인, 배포 주기를 정의한다.

### 채널 목록
1. **이메일 뉴스레터** — 수강생 + 잠재 고객 리드 육성
2. **블로그** — 네이버 블로그 + 구글 SEO/GEO, 메타 광고 노하우
3. **유튜브** — 강의 클립, 수강생 성과 사례
4. **인스타그램** — 카드뉴스, 릴스, 브랜드 인지도
5. **정보 공유 커뮤니티** — 네이버 카페, 오픈카톡방

## 3. 퍼널 × 채널 매핑

```
TOFU (인지/유입)     블로그 SEO · 유튜브 · 인스타그램 · 구글 GEO
        ↓             "이 사람 진짜 전문가네" — 콘텐츠에서 권위 인식
MOFU (신뢰/관심)     뉴스레터 · 네이버 카페 · 오픈카톡 · 유튜브 시리즈
        ↓             구독 / 카페 가입 / 뉴스레터 등록
BOFU (전환)          무료 웨비나 · 상담 신청 · 수강생 실적 데이터
        ↓             결제
RETAIN (유지)        QA 헬프데스크 · 슬랙 · 커뮤니티 (이미 구축됨)
```

## 4. 채널별 상세 기획

### 4-1. 이메일 뉴스레터

#### 퍼널 역할
MOFU → BOFU. 기존 리드(1,095명) 육성 + 블로그/카페에서 신규 구독자 유입.

#### 콘텐츠 유형
| 유형 | 설명 | 발행 주기 | 대상 |
|------|------|----------|------|
| **주간 인사이트** | 메타 광고 트렌드, 실전 팁 1개, 벤치마크 하이라이트 | 매주 화요일 | 전체 구독자 |
| **수강생 전용 뉴스레터** | 강의 보충 자료, QA 하이라이트, 과제 안내 | 격주 목요일 | 수강생만 |
| **성과 사례 레터** | 수강생 ROAS 개선 성과, 비포/애프터 | 월 1회 | 전체 구독자 |
| **웨비나 초대** | 신규 기수 모객용 무료 웨비나 안내 | 기수 시작 2주 전 | 잠재 고객 |

#### 자동화 파이프라인
```
[원본 콘텐츠 작성] → [KnowledgeService AI 요약]
        ↓
[뉴스레터 포맷 변환] — 기존 email_logs 파이프라인 활용
        ↓
[구독자 세그먼트 분기] — 수강생 / 잠재 고객 / 전체
        ↓
[Stibee or bscamp 내부 발송]
        ↓
[오픈율/클릭률 추적] → [A/B 제목 테스트 결과 저장]
```

#### 핵심 지표
- 오픈율 목표: 35%+ (업계 교육 평균 25%)
- 클릭률 목표: 5%+
- 구독자 증가: 월 100명+ (블로그/카페 CTA 통해)

#### 구현 상태: ✅ 기반 완료 (email_logs 파이프라인 + 1,095 구독자)
#### 추가 필요: 세그먼트 분기 로직, A/B 테스트, 구독자 유입 CTA 표준화

### 4-2. 블로그 (네이버 + 구글 SEO/GEO)

#### 퍼널 역할
TOFU. "메타 광고 ROAS", "상세페이지제작" 등 검색 유입의 핵심 진입점.

#### 콘텐츠 유형
| 유형 | 채널 | 글자수 | 이미지 | 발행 주기 | 키워드 티어 |
|------|------|--------|--------|----------|------------|
| **전환글 (Tier A)** | 블로그+SEO | 1,800~2,500자 | 8~12장 | 주 1회 (금) | L3~L4 (상세페이지제작, 메타광고) |
| **비교글 (Tier B)** | 블로그+SEO | 1,500~2,000자 | 7~10장 | 주 1회 (수) | L3 (광고대행사, ROAS) |
| **탐색글 (Tier C)** | 블로그+SEO | 1,200~1,800자 | 5~8장 | 주 1회 (월) | L2 (메타광고관리자, 상세페이지) |
| **정보글 (Tier D)** | 블로그 only | 800~1,200자 | 3~5장 | 추가 발행 | L1 (CPM, CPA 등 용어) |
| **GEO 마크업** | 구글 | FAQ 3~5쌍 | — | 매 글마다 | 전체 |

#### 우선 공략 키워드 (v2 로드맵 기준)
| 키워드 | 티어 | 월 검색량 | 6월 순위 목표 |
|--------|------|----------|-------------|
| 상세페이지제작 | B | 6,120 | TOP 10 |
| 메타광고 | C | 5,960 | TOP 5 |
| 상세페이지 | C | 4,640 | TOP 5 |
| ROAS | B | 3,040 | TOP 10 |
| 메타광고 세팅법 | D | ~500 | TOP 3 |
| 자사몰 마케팅 방법 | D | ~200 | TOP 1 |

#### 자동화 파이프라인
```
[bscamp 어드민 — 원본 글 작성 (마크다운)]
        ↓
[AI 포맷 변환 엔진] ← Anthropic API
        ↓
   ┌────────────────────────────────────────┐
   │  병렬 변환 (channel_distributions)      │
   │  ├─ 네이버 블로그: 2,000자+, SEO 키워드  │
   │  │   이모지 소제목, 친근+전문 톤          │
   │  ├─ 구글 인사이트: H1→H2→H3 계층         │
   │  │   Schema.org, OG 태그                 │
   │  └─ GEO 마크업: FAQ schema, HowTo,       │
   │     Speakable                            │
   └────────────────────────────────────────┘
        ↓
[Smith님 첨삭 (어드민 미리보기 모달)]
        ↓
[발행 트리거] — 즉시 or 예약
        ↓
   ① 구글 SEO (/insight/[slug]) — 즉시, 실패 없음
   ② GEO 마크업 — 같은 페이지에 schema 추가
   ③ 네이버 블로그 — OAuth API 자동 발행
        ↓
[성과 추적] — 검색 순위, 조회수, CTA 클릭률
```

#### 네이버 SEO 규칙 (C-Rank + DIA)
- **C-Rank**: 자사몰/마케팅/광고 한 분야 전문성 유지
- **DIA**: 제목 30자 이내, 키워드 앞쪽 배치, 본문 키워드 5~6회 자연 반복
- **AI 감지 우회**: 개인 경험 + 구어체 + 불규칙 문장 (Smith님 첨삭 필수)
- **이미지**: 직접 제작/촬영, alt 태그에 키워드

#### 핵심 지표
- 오가닉 트래픽: 4월 300 → 5월 1,000 → 6월 3,000 (월 방문자)
- CTA 클릭률: 5%+
- 키워드 상위노출: C/D 티어 TOP 3, B 티어 TOP 10

#### 구현 상태: 🔧 Phase 1 완료 (어드민 UI), AI 변환 엔진 + API 연동 미구현
#### 추가 필요: AI 변환 엔진, 네이버 블로그 API 연동, /insight 페이지, GEO 마크업

### 4-3. 유튜브

#### 퍼널 역할
TOFU (검색) + MOFU (시리즈 구독). 영상 콘텐츠로 전문성 시각화.

#### 콘텐츠 유형
| 유형 | 형태 | 길이 | 발행 주기 | 설명 |
|------|------|------|----------|------|
| **강의 클립** | 롱폼 | 8~15분 | 주 1회 | 강의 핵심 파트 발췌. "메타 광고 세팅 3단계" 등 |
| **수강생 성과 사례** | 인터뷰 | 10~20분 | 격주 1회 | 기존 @1bpluschool 포맷 강화. 실데이터 공개 |
| **숏폼 (Shorts)** | 세로 60초 | 15~60초 | 주 2~3회 | 강의 클립 핵심 1분, 빠른 팁, 성과 하이라이트 |
| **라이브 Q&A** | 라이브 | 30~60분 | 월 1회 | 수강생/잠재 고객 실시간 질의응답 |

#### 콘텐츠 소스
- **강의 스크립트 6주치 (2MB+)**: 이미 텍스트 존재 → AI로 영상 스크립트 재구성
- **벤치마크 데이터 (7,366rows)**: 업종별 ROAS/CPA 비교 → 데이터 시각화 영상
- **수강생 실적**: 441억 총매출, 평균 ROAS 255% → 성과 사례 소재

#### 자동화 파이프라인
```
[강의 스크립트 / 블로그 원본]
        ↓
[AI 영상 스크립트 생성] ← KnowledgeService
  - 오프닝 후크 (10초)
  - 핵심 내용 (5~12분)
  - CTA (구독, 웨비나 링크)
        ↓
[촬영 or AI 영상 생성] — Smith님 직접 촬영 권장 (신뢰도)
        ↓
[편집 자동화]
  - 자막 자동 생성 (Whisper)
  - 챕터 마커 자동 삽입
  - 썸네일 AI 생성 (제목+브랜드 컬러)
        ↓
[유튜브 업로드]
  - 제목 SEO 최적화 (키워드 + 숫자 + 결과)
  - 설명란: 타임스탬프 + bscamp 링크 + 키워드
  - 태그: 키워드 맵 기반 자동 태깅
  - 카드/엔드스크린: 관련 영상 + 웨비나 링크
        ↓
[숏폼 자동 추출]
  - 롱폼 → 핵심 60초 클립 3개 자동 컷
  - Shorts + 인스타 릴스 동시 배포
```

#### 핵심 지표
- 구독자: 6개월 내 1,000명 (수익화 기준)
- 평균 조회수: 롱폼 500+, 숏폼 2,000+
- bscamp 유입 전환: 설명란 링크 클릭률 3%+

#### 구현 상태: ❌ 미구현 (계정만 존재, @1bpluschool)
#### 추가 필요: 영상 스크립트 AI 생성, 편집 워크플로, 업로드 자동화, 숏폼 추출

#### 우선순위: Phase 3 (검색형 채널 안정화 후)
#### 단, 숏폼 추출은 블로그 콘텐츠 기반으로 Phase 2에서 병행 가능

| 유형 | 형태 | 발행 주기 | 설명 |
|------|------|----------|------|
| **카드뉴스** | 캐러셀 (5~10장) | 주 2회 | 블로그 글 핵심 요약. "메타 광고 ROAS 올리는 5가지 방법" |
| **릴스** | 세로 영상 (15~60초) | 주 2~3회 | 유튜브 숏폼 재활용 + 인스타 네이티브 클립 |

## 6. "1원본 → 5채널" 변환 흐름

### 콘텐츠 생산 원칙
- **원본은 1개**: bscamp 어드민에서 마크다운으로 작성 (1,500~2,500자)
- **변환은 AI**: Anthropic API로 채널별 톤/형태/길이 자동 변환
- **첨삭은 Smith님**: AI 감지 방지 + 실제 경험 삽입 (블로그만 필수)
- **배포는 자동**: 채널별 API or 큐 기반 병렬 배포

### 변환 매트릭스
| 원본 (1,500~2,500자) | → 블로그 | → 카페 | → 뉴스레터 | → 유튜브 스크립트 | → 인스타 카드 |
|----------------------|----------|--------|-----------|-----------------|-------------|
| 톤 | 친근+전문 | 구어체 | 핵심 요약 | 대화체 | 핵심 문장 |
| 길이 | 2,000자+ | 800~1,200자 | 500~800자 | 8~15분 분량 | 5~8 카드 |
| 형태 | HTML, 이모지 소제목 | 텍스트, 질문 마무리 | 이메일 HTML | 스크립트+자막 | 이미지+텍스트 |
| 키워드 | SEO 최적화 | 자연 삽입 | 제목만 | 태그 | 해시태그 |
| CTA | bscamp 링크 | 댓글 유도 | 클릭 버튼 | 설명란 링크 | 프로필 링크 |
| 배포 시차 | 즉시 (D+0) | D+1~2 | D+1 (화) | D+3~4 (편집 후) | D+1 |
