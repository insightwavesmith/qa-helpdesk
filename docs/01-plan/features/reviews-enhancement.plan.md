# 수강후기 탭 강화 Plan

> 작성: 2026-03-01

## 배경
- 후기 작성 시 제목+본문+이미지만 입력 가능, 기수/카테고리/별점 없음
- 전체 후기가 시간순으로만 표시, 필터 없음
- 유튜브 수료생 인터뷰 시리즈가 있지만 QA Helpdesk에서 볼 수 없음
- 관리자가 후기 삭제만 가능, 등록/수정/고정 불가

## 범위
- B1: 후기 작성폼에 기수/카테고리/별점 추가
- B2: 후기 목록 필터링 + 정렬
- B3: 유튜브 후기 영상 임베드 (관리자 전용)
- B4: 관리자 후기 관리 페이지 (/admin/reviews)

## 범위 외
- 기존 후기 데이터 마이그레이션
- 댓글/좋아요 기능
- 검색 기능
- 무한 스크롤
- 수강생 후기 수정 기능
- 후기 승인 프로세스

## 성공 기준
1. 후기 작성 시 기수, 카테고리, 별점 선택 가능
2. 후기 목록에서 기수별/카테고리별 필터 + 최신순/별점순 정렬 가능
3. 관리자가 유튜브 URL 포함 후기 등록 가능, 상세 페이지에서 영상 임베드
4. /admin/reviews 페이지에서 전체 후기 관리 + 고정 기능
5. npm run build 성공
