# 답변 승인 시 솔라피 카카오 알림톡 발송 — Plan

## 1. 개요
관리자가 답변을 승인하면, 질문 작성자에게 카카오 알림톡으로 "답변 완료" 알림을 자동 발송한다.

## 2. 배경
- 수강생이 답변이 달렸는지 매번 확인하러 접속할 필요 없음
- 카톡 알림으로 답변 확인율 향상

## 3. 범위
- 솔라피 HMAC 인증 + 알림톡 발송 함수 (`src/lib/solapi.ts` 신규)
- `approveAnswer()` 내에서 질문 작성자 phone 조회 → 알림톡 발송 (fire-and-forget)
- 환경변수: `SOLAPI_API_KEY`, `SOLAPI_API_SECRET`

## 4. 범위 외
- 솔라피 npm SDK 사용 안 함 (fetch + HMAC 직접 구현)
- 대체 SMS 발송 없음 (`disableSms: true`)
- 발신번호(from) 불필요 (카카오 채널이 발신자)

## 5. 성공 기준
- [ ] 답변 승인 시 질문 작성자에게 알림톡 발송
- [ ] 전화번호 없는 수강생은 조용히 스킵
- [ ] 발송 실패해도 승인 자체는 정상 처리 (fire-and-forget)
- [ ] `tsc --noEmit` 통과
- [ ] 기존 승인 기능 영향 없음
