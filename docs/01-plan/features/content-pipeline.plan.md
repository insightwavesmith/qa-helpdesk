# 콘텐츠 파이프라인 계획서

## 1. 개요
- 2개 파이프라인: 외부(크롤링) + 내부(Smith님 직접 입력)
- 모찌가 글+이미지 작성 → Smith님 수정 → 원클릭 발행/발송
- 정보공유(긴 글) + 뉴스레터(함축)를 1:1로 생성

## 2. 범위

### 포함
- 외부 파이프라인: 크롤링 → 가공 → draft 저장
- 내부 파이프라인: Smith님 소스(PDF/URL/글/지시) → 가공 → draft 저장
- AI 이미지 생성 (헤더 + 인포그래픽)
- 뉴스레터 발송 플로우 (leads 전원)
- 뉴스레터 구독 페이지

### 제외
- 인라인 편집 UI (별도 기능)
- 크롤링 자동화 크론 (Phase 4)

## 3. 성공 기준
