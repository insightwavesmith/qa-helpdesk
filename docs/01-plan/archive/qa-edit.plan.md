# Q&A 글 수정 기능 — Plan

## 요약
수강생이 Q&A 질문을 수정할 수 있는 기능 추가. 기존 글 작성 UI 재활용.

## 범위
- Q&A 질문 수정 (제목, 본문, 카테고리, 이미지)
- 본인 글 + 관리자(admin/assistant) 수정 가능
- 기존 NewQuestionForm 컴포넌트 재활용

## 범위 외
- 답변(answer) 수정 (별도 TASK)
- Q&A 목록 페이지 변경
- 삭제 기능
- 에디터 라이브러리 변경

## 성공 기준
- 본인 질문에 "수정" 버튼 표시
- 수정 버튼 → 기존 글 작성 UI로 편집 모드 전환
- 제목/본문/카테고리/이미지 수정 가능
- 관리자는 모든 글 수정 가능
- npm run build 성공
