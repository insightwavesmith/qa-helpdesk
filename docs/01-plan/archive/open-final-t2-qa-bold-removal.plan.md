# T2: QA 답변 볼드(**) 제거 — Plan

## 요구사항
AI 자동 답변에서 `**볼드**` 마크다운 강조를 제거하고, 평문으로만 작성하도록 프롬프트를 수정한다.

## 배경/맥락
- QA 답변은 `src/lib/knowledge.ts`의 `QA_SYSTEM_PROMPT`에 의해 생성
- 현재 프롬프트: "**굵기 강조**만 허용. 핵심 키워드나 숫자에만 사용."
- 결과: `**총가치 = 입찰가 × 추산행동률 + 광고품질**` 같은 볼드 포함 답변 생성
- 실무 선배 답변 톤에 볼드 강조는 블로그 글처럼 보여서 어울리지 않음
- 이전 수정 이력: `f2-qa-answer-humanize` (2026-03-04) + `b1b2-t2fix-t4fix-bugfix` (2026-03-05) + `qa-curation-quality-t1t6` (2026-03-05)에서 프롬프트 개선했으나, 볼드 규칙은 "허용"으로 남아 있음

2. 기존 답변은 영향 없음 (이미 DB에 저장됨)
3. tsc --noEmit + next lint + npm run build 통과
