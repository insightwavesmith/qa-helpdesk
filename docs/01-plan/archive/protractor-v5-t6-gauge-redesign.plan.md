# T6. 게이지 목업 디자인 일치 — Plan

> 작성일: 2026-03-04
> 스프린트: 총가치각도기 v5
> 목업 참조: docs/mockups/protractor-v5.html

## 1. 개요

- **기능**: `TotalValueGauge` 컴포넌트의 SVG 게이지와 서브점수 카드를 목업과 일치시키도록 재디자인
- **해결하려는 문제**: 현재 게이지가 파스텔 색상 + 시계바늘 + 파란 진행 아크 + 0/50/100 마커 + 텍스트형 파트 점수 바를 사용하여 목업과 시각적으로 다름
- **목업 기준**: protractor-v5.html의 `.gauge-wrap` + `.grade-cards` 섹션

## 2. 현재 상태 vs 목표 상태

| 항목 | 현재 (TotalValueGauge.tsx) | 목표 (목업 기준) |
|------|--------------------------|----------------|
| 등급 구간 색상 | 파스텔 (#fca5a5 빨강, #fde68a 노랑, #86efac 초록) | 선명한 (#ef4444 빨강, #eab308 노랑, #22c55e 초록) |
| 포인터 | 중심에서 뻗는 line + pivot circle | 호(arc) 위 원형 도트 (circle) |
| 파란 진행 아크 | 있음 (score > 0일 때) | 제거 |
| 0/50/100 수치 마커 | 있음 | 제거 |
| 배경 회색 호 | 없음 | #e2e8f0 추가 |
| 서브점수 | 텍스트 바 (`PartScoreBar`): "기반점수 A" 형태 | 독립 카드: 점수 숫자(72) + 등급 배지(A) |
| 게이지 내부 표시 | SVG 내 score + 등급 텍스트 | 게이지 아래 별도 영역: score 큰 숫자 + 등급 배지 + "N일 기준 · 전체 광고 합산" |
| 서브점수 카드 위치 | 게이지 왼쪽 하단 (수직 나열) | 게이지 오른쪽 (flex 카드 3개 수직 나열) |

## 3. 핵심 요구사항

### 기능적 요구사항
- FR-01: 등급 구간 색상 → 선명한 빨강/노랑/초록 (#ef4444/#eab308/#22c55e)
- FR-02: 포인터 → 호 위 원형 도트로 교체 (line + pivot 제거)
- FR-03: 파란 진행 아크 제거
- FR-04: 0/50/100 수치 마커 제거
- FR-05: 배경 회색 호 추가 (#e2e8f0, stroke-width=16)
- FR-06: 서브점수 → 독립 카드 3개 (기반점수/참여율/전환율): 점수 숫자 + 등급 배지
- FR-07: 중앙 점수 표시: 큰 숫자 + N등급 배지 + 기간 레이블 (게이지 아래)

### 비기능적 요구사항
- 점수 계산 로직(t3-engine) 변경 금지
- 타겟중복 분석 섹션 변경 금지
- 콘텐츠 탭 변경 금지
- `showMetricCards` prop 동작 유지 (하위 호환)
