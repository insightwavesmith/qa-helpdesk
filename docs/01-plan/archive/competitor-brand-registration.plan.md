# 경쟁사 분석기 브랜드 등록 개선 Plan

> 작성일: 2026-03-07

## 배경

현재 브랜드 모니터링 등록 시 텍스트 입력만 가능하여, 실제 Meta 페이지와의 매칭이 부정확할 수 있다. Meta Ad Library API를 활용해 페이지를 검색하고 드롭다운에서 선택하는 방식으로 개선하여 정확한 page_id 연결을 보장한다.

## 범위

| 태스크 | 설명 | 의존성 |
|--------|------|--------|
| **T1** | 브랜드 검색 API (`/api/competitor/pages`) | 기존 `searchMetaAds` 활용 |
| **T2** | 브랜드 등록 다이얼로그 개선 (검색+드롭다운+칩) | T1 |
| **T3** | 모니터링 카드에 페이지 로고 표시 | 없음 |

## 성공 기준

- [ ] `npm run build` 성공
- [ ] T1: `/api/competitor/pages?q=키워드` 호출 시 고유 페이지 목록 반환
- [ ] T2: 검색 입력 → debounce 300ms → 드롭다운(프로필 이미지+페이지명) → 선택 → 등록
- [ ] T2: 기존 텍스트 직접 입력도 유지 (pageId null)
- [ ] T3: pageId 있는 모니터에 Graph API 프로필 이미지 표시, fallback 아바타
- [ ] bscamp 디자인 시스템 적용
