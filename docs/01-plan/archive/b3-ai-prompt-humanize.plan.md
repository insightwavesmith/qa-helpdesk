# B3. AI 프롬프트 humanize — Plan

> 작성: 2026-03-02
> 선행 작업: T5(content-ai-prompt) — completed (Match Rate 95%)

## 1. 개요
- **기능**: 정보공유 AI 생성 글의 문체를 자연스럽게 개선 (AI틱한 느낌 제거)
- **해결하려는 문제**: T5에서 구조적 프롬프트 개선(3줄요약/숫자블록/체크리스트/구분선)을 했으나, 문체 자체가 여전히 로봇 느낌
  - "-었요. -요" 반복 어미
  - 모든 문장이 비슷한 길이
  - "매우 중요합니다", "필수적입니다" 같은 AI 상투어 반복
- **수정 대상**: `src/app/api/admin/curation/generate/route.ts` (system prompt)

## 2. 핵심 요구사항

### 기능적 요구사항
- FR-01: **문장 끝 다양화** — ~요, ~다, ~죠, ~거든요, ~네요, ~데요 섞어서 쓰기. 같은 어미 3번 연속 금지.
- FR-02: **문장 리듬** — 짧은 문장(10자 이내)과 긴 문장(40자+) 번갈아.
- FR-03: **의문문/감탄문** — 섹션당 1~2개. "왜 이렇게 될까요?" "진짜 됩니다."
- FR-04: **구어체 전환어** — "사실", "솔직히", "그런데 말이죠" 적절히.
- FR-05: **AI 상투어 금지** — "매우 중요합니다", "필수적입니다", "핵심입니다", "반드시" 등.
- FR-06: **경험담 톤** — 교육 현장에서 실제로 본 것처럼.

### 비기능적 요구사항
- AI 모델 변경 금지
- RAG 로직 변경 금지
- 글 구조(훅→도입→목차→본론→마치며) 변경 금지
- 최소 글자 수(4,000자) 변경 금지
