# TASK: 총가치각도기 UI 목업 프로토타입

## 고객 관점
총가치각도기 UI/UX 업데이트 계획서(`docs/protractor-uiux-update-plan.md`)에 나온 4개 탭의 **실제 UI가 어떻게 보일지** 인터랙티브 목업을 만들어야 한다. 기획자가 "이렇게 생긴 걸 만들자"고 보여주는 프로토타입이다.

## 산출물
**단일 HTML 파일** → `/Users/smith/projects/mozzi-reports/public/reports/plan/2026-03-19-protractor-ui-mockup.html`

## 요구사항

### 디자인
- **다크 테마** (배경 #0f172a, 카드 #1e293b, 텍스트 white/gray)
- Tailwind CSS CDN 사용
- **4개 탭 전환 가능** (JavaScript):
  1. 대시보드 탭
  2. 소재 분석 탭
  3. 랜딩페이지 탭
  4. 경쟁사 분석 탭

### 각 탭별 구현 내용 (계획서 3.1~3.4 참조)

**1. 대시보드 탭:**
- 상단: 5개 핵심 지표 카드 (총가치점수, ROAS, CPA, CTR, 전환율) — 숫자 + 전월대비 증감
- 중단: 3부문 점수 카드 (기반/참여/전환) — 프로그레스 바 + 색상
- 하단: 소재 랭킹 테이블 (순위, 썸네일, 점수, ROAS, 상태 배지)

**2. 소재 분석 탭:**
- 좌측(38%): 소재 썸네일 + 기본 정보 + L4 점수 원형 배지 + L1 태그
- 우측(60%): 요소별 벤치마크 패턴 비교 테이블 (🟢🟡🔴) + AI 개선 제안
- 하단: 히트맵 섹션 (좌: 소재 위에 히트맵 오버레이 그라디언트, 우: 시선 집중도 테이블 + 동선 비교)

**3. 랜딩페이지 탭:**
- 좌측(38%): URL 입력 + 폰 목업 LP 스크린샷 + 구조 체크리스트 (✅❌⚠️)
- 우측(60%): LP 종합 점수 원형 배지 + 벤치마크 LP 패턴 비교 + AI 개선 제안

**4. 경쟁사 분석 탭:**
- 기존: 키워드 검색 + 광고 카드 그리드
- 신규: "내 소재와 비교" 버튼 → 2-컬럼 비교 패널 (내 소재 vs 경쟁사)

### 데이터
- **전부 더미 데이터** (실제 API 호출 없음)
- 화장품 브랜드 광고 맥락에 맞는 리얼한 더미값
- 차트는 CSS 그라디언트/바로 구현 (Chart.js 등 외부 라이브러리 최소화)

### 기술
- 단일 HTML, inline CSS/JS
- Tailwind CDN: `<script src="https://cdn.tailwindcss.com"></script>`
- 탭 전환 JavaScript
- 반응형 (데스크탑 기준, 모바일은 스택)
- 히트맵은 CSS gradient overlay로 시뮬레이션
