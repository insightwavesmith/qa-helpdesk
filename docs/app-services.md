# 자사몰사관학교 앱 서비스 레퍼런스

> Claude가 수강생 질문에 답변할 때 참조하는 앱 기능 요약.
> "우리 서비스로 해결 가능한가?" 먼저 확인 후 답변.

---

## 서비스 한눈에 보기

| 서비스 | 한줄 설명 | 관련 질문 키워드 |
|--------|----------|----------------|
| 브라우저 트래커 | 고객 행동 자동 추적 | 추적, 이벤트, Mixpanel |
| Meta CAPI | 서버사이드 전환 전송 | 전환API, 쿠키차단, 픽셀보완 |
| EMQ 모니터링 | 이벤트 품질 점수 확인 | 품질점수, EMQ, 매칭률 |

| 과거 데이터 Import | Mixpanel에 주문 이력 전송 | 과거데이터, 히스토리 |

---

## 1. 브라우저 트래커

**목적:** 고객 구매 여정 실시간 추적

**추적 이벤트:**
- 페이지/상품 조회
- 장바구니 추가/제거
- 결제 시작 → 결제 수단 선택 → 구매 완료
- 회원가입, 검색

**자동 수집:** 상품 정보, UTM, 디바이스, 유입 경로

**연동:** Mixpanel (분석), Meta Pixel (광고)

---

## 2. Meta Conversion API (CAPI)

**목적:** 쿠키 차단에도 전환 추적 유지

**핵심 기능:**
- 서버 → Meta 직접 전송 (브라우저 우회)
- 픽셀과 중복 제거 (event_id 기반)
- 회원 정보 자동 해싱 전송 (타겟팅 정확도 ↑)
- 비회원 → 회원 전환 시 과거 이벤트 재전송
- 실패 시 자동 재시도 (1시간마다, 최대 3회)

**전송 이벤트:** ViewContent, AddToCart, InitiateCheckout, AddPaymentInfo, Purchase, CompleteRegistration

**설정:** 픽셀 ID, Access Token, 테스트 모드

---

## 3. EMQ 모니터링

**목적:** Meta 이벤트 품질 확인

**확인 가능:**
- 이벤트별 EMQ 점수 (10점 만점)
- 어떤 고객 정보가 매칭에 기여하는지
- CAPI 활성화 전/후 비교

**활용:** 점수 낮으면 고객 정보 보강 필요 신호

---

## 4. 광고 데이터 수집

**목적:** Meta 광고 성과 자동 수집

**수집 데이터:**
- 캠페인/광고세트/광고별 성과
- 노출, 클릭, 비용, 전환

**동작:** 매일 자동 (D-2~D-1), 수동 트리거 가능

**활용:** ROAS 분석, Cafe24 주문과 연계 검증

---

## 5. 카탈로그 동기화

**목적:** Cafe24 상품 → Meta 카탈로그 자동 연동

**동기화 정보:** 상품명, 설명, 가격, 할인가, 이미지, 재고, 카테고리, URL

**동작:** 매일 새벽 자동, 수동 가능

**활용:**
- Meta Shop (인스타/페북 숍)
- DPA (리타겟팅)
- Advantage+ 카탈로그 광고

---

## 6. 체험단 캠페인

**목적:** 체험단 마케팅 자동화

**운영 흐름:**
1. 캠페인 생성 (상품, 인원, 기간, 조건)
2. 신청 접수 (공개 페이지)
3. 선정/발표 (일괄 처리)
4. 리뷰 수집
5. 데이터 CSV 추출

**관리:** 상태 관리, 일괄 승인/거절, CSV 내보내기

---

## 7. 과거 데이터 Import

**목적:** Mixpanel에 과거 주문 전송

**기능:** 날짜 범위 지정, 백그라운드 처리

**활용:** 서비스 도입 전 고객 분석, 세그먼트 생성, 리텐션 기준 확보

---

## 8. 어드민 기능

**고객사 관리:** 목록 조회, 설정 확인/수정

**모니터링:** CAPI 로그, 일일 리포트 (실패율 5% 초과 경고)

**운영:** Import/수집 수동 트리거, 화이트리스트

## 질문 → 서비스 매핑 가이드

| 수강생 질문 | 먼저 확인할 서비스 | 일반론 대신 이렇게 |
|------------|------------------|------------------|
| "전환 추적 안 돼요" | CAPI 설정, EMQ 점수 | "CAPI 연동 상태 확인해봐요" |
| "광고 성과 안 나와요" | EMQ, 광고 데이터 | "EMQ 점수 먼저 확인해보세요" |
| "리타겟팅 하고 싶어요" | 카탈로그 동기화 | "카탈로그 동기화 되어있나요?" |
| "컨텐츠 어떻게 수급해요" | 체험단 캠페인 | "체험단 프로세스 활용해보세요" |
| "고객 분석 하고 싶어요" | Mixpanel, 과거 Import | "Mixpanel에서 퍼널 분석 가능해요" |
| "상품광고 어떻게 해요" | 카탈로그, DPA | "DPA 세팅되어있나요?" |
