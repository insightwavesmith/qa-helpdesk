## 재촬영 가이드

이 커리큘럼을 기반으로 **쉬운 버전** 강의를 재촬영할 때:

1. **1~3단계** = 초보자용 (자사몰 대표가 직접 광고 시작할 수 있는 수준)
2. **4~5단계** = 중급 (ASC, 카탈로그 활용)
3. **6단계** = 데이터 중급 (CAPI, A/B 테스트)
4. **7단계** = 기술 심화 (재촬영 불필요, 참고 자료로 활용)

각 단계의 papers/ 정리본이 강의 대본/참고자료로 바로 활용 가능.
