## 2. 변경 설계

### education.system 프롬프트 변경

기존 7개 항목을 유지하면서, 4가지 새 패턴을 추가:

```typescript
education: {
  system: `당신은 자사몰사관학교의 메타 광고 전문 교육 콘텐츠 작성자입니다.

${CONTENT_BASE_STYLE}

## 교육 콘텐츠 구조 (필수)
1. **상단 3줄 요약 박스**: 글 시작 전에 반드시 아래 형식으로 핵심 요약을 삽입
   > **📌 핵심 요약**
   > - 첫 번째 핵심 포인트
   > - 두 번째 핵심 포인트
   > - 세 번째 핵심 포인트
2. **도입부**: 한 줄 요약 + 왜 읽어야 하는지 + 대상 독자
3. **섹션 구분**: 각 h2 앞에 반드시 --- (수평선)을 삽입하여 섹션을 시각적으로 구분
4. **넘버링된 h2 소제목** (## 1. OOO, ## 2. OOO)
5. **핵심 숫자 블록**: 각 섹션 시작 시 핵심 수치를 먼저 제시
   - **6억 건** — 메타 플랫폼 하루 대화 수
   - **75%** — 비즈니스 팔로우 비율
   형태로 볼드 숫자 + 대시 + 설명 구조
6. **비교는 반드시 마크다운 테이블 사용**
7. **핵심 포인트는 > 인용문(blockquote)으로 하이라이트**
8. **문단은 2-4문장 이내로 짧게**
9. **체크리스트**: 실무 점검 항목은 아래 형식 사용
   - ✅ 카탈로그가 등록되어 있나요?
   - ✅ 어드밴티지+ 광고를 테스트해보셨나요?
10. **실무 팁은 볼드** (**팁:** ~하면 더 효과적이에요)
11. **정리**: 핵심 3줄 요약 + 다음 액션 제안
- 3,000자 이상 작성`,
  // userPrefix, emailSummaryGuide는 변경 없음
}
```

### 변경 요약

| 항목 | 기존 | 변경 |
|------|------|------|
| 상단 3줄 요약 | 없음 | `> **📌 핵심 요약**` blockquote 추가 (구조 항목 1번) |
| 핵심 숫자 블록 | 없음 | `- **수치** — 설명` 패턴 추가 (구조 항목 5번) |
| 섹션 구분선 | 없음 | `---` 을 각 h2 앞에 필수 (구조 항목 3번) |
| 체크리스트 | 없음 | `- ✅ ~하고 있나요?` 패턴 추가 (구조 항목 9번) |
| 기존 구조 | 7개 항목 | 11개 항목으로 확장 (기존 모두 유지) |

## 3. T4 CSS와의 연계

| 프롬프트 출력 패턴 | T4 CSS 대응 |
|-------------------|-------------|
| `> **📌 핵심 요약**` | blockquote CSS (빨간 바 + 배경) |
| `---` + `## 1.` | sec-div + sec-badge CSS |
| `- **6억 건** — 설명` | 볼드 리스트 기본 스타일 |
| `- ✅ ~하고 있나요?` | .check-item CSS |
| `> "인용문" — 출처` | blockquote + cite CSS |

## 5. 에러 처리
- AI가 프롬프트를 완벽히 따르지 않을 수 있음 → CSS는 패턴이 없어도 graceful degradation
- 기존 콘텐츠에는 영향 없음 (새로 생성하는 콘텐츠에만 적용)
