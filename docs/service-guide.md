# 자사몰사관학교 서비스 가이드

이 문서는 정보공유 글 작성 및 QA 답변 시 참조되는 서비스 컨텍스트다. 외부 콘텐츠에서 언급되는 도구/서비스를 우리 서비스에 맞게 대체할 때 사용한다.

---

## 우리가 쓰는 도구

- **매출/행동 분석**: Mixpanel (GA4, 구글 애널리틱스 아님)
- **광고 플랫폼**: Meta 광고 관리자 (구글 애즈 아님)
- **전환 추적**: Meta Pixel + Conversion API (CAPI) 서버사이드
- **성과 대시보드**: 총가치각도기 (엑셀/노션/수동 리포트 아님)
- **쇼핑몰 플랫폼**: Cafe24
- **상품 카탈로그**: Meta 카탈로그 자동 동기화 (수동 업로드 아님)
- **이메일 마케팅**: Stibee

## 데이터 수집

- 광고 데이터: 매일 KST 03:00 자동 수집 → daily_ad_insights
- Mixpanel 매출: 매일 KST 03:30 자동 수집 → daily_mixpanel_insights
- 벤치마크: 매주 화요일 KST 02:00 자동 수집
- 수강생이 직접 데이터를 기록하거나 엑셀에 정리할 필요 없다. 총가치각도기 대시보드에서 자동으로 확인 가능.

## 총가치각도기

수강생의 광고 성과를 자동 분석하는 대시보드다.

- T3 점수: 기반점수(40%) + 참여율(30%) + 전환율(30%)
- 등급: A~F (벤치마크 대비 상대 평가)
- 핵심 지표: ROAS, CPC, CTR, CPM, 3초 시청률
- 기간별 조회: 7일/30일/전체
- 광고 유형: VIDEO, CATALOG(카탈로그/DPA)

## 교육 서비스 구조

- 기수제 운영 (현재 5기)
- 3주 커리큘럼, 온라인 교육
- 메타 광고 전문 교육 (구글/네이버 광고 아님)
- 수강생 대상: Cafe24 자사몰 운영자
- 강사: Smith (메타 광고 10년차, 메타 공식파트너)

## 자사몰사관학교 카페24 앱

자사몰사관학교는 Cafe24 쇼핑몰을 위한 마케팅/분석 통합 솔루션이다. 쇼핑몰에 스크립트 하나를 설치하면 고객 행동 추적, Meta 광고 최적화, 상품 카탈로그 동기화, 체험단 캠페인까지 한번에 관리할 수 있다.

### 고객 행동 추적 (브라우저 트래커)

쇼핑몰 페이지에 자동으로 삽입되는 트래커 스크립트가 고객의 구매 여정을 실시간으로 추적한다. 추적 이벤트: 페이지 조회, 상품 조회, 상품 클릭, 장바구니 추가/제거, 결제 시작, 구매 완료, 회원가입 완료, 검색 등. 자동 수집 정보: 상품 정보, UTM 파라미터, 디바이스/브라우저 정보, 유입 경로. 연동 대상: Mixpanel(행동 분석), Meta Pixel(전환 추적).

### Meta Conversion API (CAPI)

브라우저 쿠키 차단으로 Meta Pixel만으로는 전환 추적이 부정확해지는 문제를 해결한다. 서버에서 직접 Meta에 전환 데이터를 보내는 Server-Side 방식. 핵심: 서버 사이드 이벤트 전송, 중복 제거(event_id), 고객 정보 자동 보강(해싱 후 전송), 비회원→회원 전환 매칭, 웹훅 기반 구매 이벤트, 자동 재시도(최대 3회). 전송 이벤트: ViewContent, AddToCart, InitiateCheckout, AddPaymentInfo, Purchase, CompleteRegistration.

### 이벤트 품질 점수 (EMQ) 모니터링

Meta가 수신한 이벤트의 품질을 측정하는 EMQ 점수를 대시보드에서 확인 가능. 이벤트별 EMQ 점수(10점 만점), 픽셀 간 비교, 매칭 기여 정보 확인. EMQ 점수가 높을수록 전환 최적화와 타겟팅이 정확해진다.

### Meta 광고 데이터 수집

Meta 광고 계정의 성과 데이터를 자동으로 수집하여 저장. 캠페인/광고 세트/광고별 성과, 노출수, 클릭수, 비용, 전환 데이터. 매일 자동 실행(D-2~D-1 수집). 필요시 수동 수집 가능.

### Meta 상품 카탈로그 동기화

Cafe24 상품 데이터를 Meta 카탈로그에 자동 동기화. 상품명, 설명, 가격, 할인가, 이미지, 재고, 카테고리 매핑, URL. 매일 새벽 자동 전체 동기화. Meta Shop, DPA(리타겟팅), Advantage+ 카탈로그 광고에 활용.

### 체험단 캠페인 관리

신제품 홍보나 리뷰 수집을 위한 체험단 캠페인 관리. 캠페인 생성 → 신청 접수 → 선정/발표 → 리뷰 수집 → CSV 내보내기.

### 과거 데이터 Import

Mixpanel에 과거 주문 데이터를 일괄 전송. 서비스 도입 이전 고객 행동 분석, 기존 고객 세그먼트 생성, 리텐션 분석 기준 데이터 확보.

### 멀티샵 지원

Cafe24 멀티샵 기능 지원. 샵별 독립된 Mixpanel/Meta Pixel/카탈로그 설정.

## 외부 콘텐츠 참조 시 주의사항

외부 블로그/뉴스에서 아래 도구가 언급되면 우리 서비스 맥락으로 대체한다:
- GA4, 구글 애널리틱스 → Mixpanel
- 엑셀/노션에 데이터 기록 → 총가치각도기 대시보드에서 자동 확인
- 수동 리포트 작성 → 매일 자동 수집
- 구글 태그 매니저(GTM) → 자사몰사관학교 트래커 스크립트
- 구글 애즈 → Meta 광고
- Shopify → Cafe24
