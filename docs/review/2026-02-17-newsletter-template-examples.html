### INSIGHT
2025년 메타 안드로메다 알고리즘은 AI 기반 광고-유저 매칭을 **100배 가속화**했어요. 이제 넓은 타겟 + 다양한 소재 조합이 핵심 전략이에요. 기존의 좁은 타겟팅보다 **어드밴티지+ 오디언스**를 활용한 넓은 그물 전략이 평균 **ROAS 30% 이상** 높은 성과를 보여주고 있어요.
> 💡 소재 3~5개를 광고세트당 세팅하고, 2~3주마다 리프레시하는 게 베스트 프랙티스예요.

### KEY POINT
✅ **어드밴티지+ 캠페인 세팅** — ASC(어드밴티지+ 쇼핑 캠페인)를 활용하면 AI가 자동으로 최적 조합을 찾아요. CBO 설정으로 예산도 자동 분배돼요.
✅ **픽셀 + CAPI 듀얼 추적** — 픽셀만으로는 데이터 정확도가 **60~70%**에 불과해요. 전환 API를 함께 세팅하면 **95%+** 정확도를 확보할 수 있어요.
✅ **크리에이티브 다양화 전략** — 정적 이미지, 카루셀, 릴스 영상을 혼합 세팅하세요. AI가 유저별 최적 포맷을 자동 선택해요.

Case Study — 템플릿 C TEMPLATE_C

1. AI 생성 email_summary 마크다운

화장품 자사몰 월매출 300만 원에서 1,200만 원으로 성장한 스킨블리스 대표님의 이야기예요.

### 성과
#### 핵심 지표 변화
| 지표 | Before | After |
| --- | --- | --- |
| 월 매출 | 300만 원 | 1,200만 원 |
| ROAS | 1.2 | 4.8 |
| 일 평균 주문 | 3건 | 15건 |
| 광고 소재 수 | 2개 | 12개 |

#### 전환 성과
| 항목 | 수치 |
| --- | --- |
| 수강 기간 | 8주 |
| CPA 감소율 | -62% |
| 재구매율 향상 | +45% |

### INTERVIEW
> "처음엔 광고비만 태우고 있다고 생각했는데, ASC 캠페인 세팅 방법을 배우고 나서 완전히 달라졌어요. 특히 크리에이티브 다양화 전략이 제일 큰 전환점이었어요." — 수강생 김서영님

### 핵심 변화
✅ **타겟팅 전략 전환** — 좁은 관심사 타겟 → 어드밴티지+ 오디언스로 전환. AI가 최적 유저를 찾아주면서 CPA가 62% 감소했어요.
✅ **크리에이티브 다양화** — 단일 이미지 2개 → 정적+카루셀+릴스 12개로 확대. 소재 다양성이 알고리즘 최적화의 핵심이었어요.
✅ **데이터 추적 강화** — 픽셀만 사용 → 픽셀+CAPI 듀얼 설정. 전환 데이터 정확도가 95%로 올라가면서 최적화 속도가 빨라졌어요.

항목현재재설계 후
본문 구조 단일 content-body-text-1에 전체 HTML 배너키별 2개 row (배너 이미지 + 콘텐츠)
