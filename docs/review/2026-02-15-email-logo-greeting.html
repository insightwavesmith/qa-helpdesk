모찌에게 드리는 요약: Smith님 피드백을 반영해서 3가지를 바꿉니다.

  1. 1.
    로고 배경이 투명해져요.
    지금 로고(v3)는 빨간 배경이 깔려 있어서, 이메일의 흰 바탕과 어울리지 않았어요. 새 로고(v4)는 투명 배경이라 흰 이메일 위에서 자연스럽게 보입니다. 빨간 마크 + "자사몰사관학교" 검정 텍스트만 깔끔하게 표시돼요.
  2. 2.
    "안녕하세요! 대표님" 인사말이 삭제돼요.
    모든 이메일 템플릿에서 "안녕하세요! 대표님, 자사몰사관학교의 스미스코치입니다." 문구가 사라집니다. 로고 다음에 바로 본문 내용이 나오게 됩니다.
  3. 3.
    체크리스트가 모바일에서 깨지지 않아요.
    이메일 본문에 체크리스트(✓ 항목들)가 들어갈 때, 모바일 화면에서 텍스트가 잘리거나 밀리는 문제가 있었어요. 테이블 구조를 간소화해서 375px 작은 화면에서도 정상 표시되도록 수정합니다.

4 검증 체크리스트

# 검증 항목 상태
1 npm run build 성공 대기
2 4개 템플릿 Unlayer 에디터에서 v4 로고 투명 배경 표시 확인 대기
3 Template A/B/C에서 "안녕하세요! 대표님" 인사말 안 보임 대기
4 Default 템플릿: 인사말 없이 placeholder만 표시 대기
5 기존 배너/프로필카드/CTA 버튼 렌더링 정상 대기
6 테스트 이메일 발송 → Gmail에서 로고가 흰 배경과 자연스럽게 표시 대기
7 테스트 이메일에서 "스미스코치입니다" 텍스트 안 보임 대기
T3. 체크리스트 모바일 반응형
8 모바일(375px)에서 체크리스트 텍스트 밀림/잘림 없이 정상 표시 대기
9 번호 카드(bold ✓) 모바일에서 번호/텍스트 정상 배치 대기
10 Gmail 모바일 앱에서 체크리스트 렌더링 정상 대기
