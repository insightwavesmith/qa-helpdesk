## 사용 시 주의사항

1. Meta 로고는 **공식 가이드라인**을 반드시 준수하세요
2. Facebook 로고의 색상 변경, 3D 가공, 윤곽선 추가 등은 **금지**됩니다
3. Instagram은 'Insta', 'gram' 등으로 축약하지 마세요
4. 블로그 이미지는 교육/참고 목적으로만 사용하세요
