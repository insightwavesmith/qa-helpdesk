# Playwright Pro

Playwright 기반 E2E 테스트 생성 스킬.

## 기능

- 페이지 네비게이션 테스트
- 폼 입력/제출 테스트
- 인증 플로우 테스트
- API 응답 검증
- 스크린샷 비교
- 모바일/데스크톱 반응형 테스트

## 사용법

CTO/QA가 `/playwright` 호출하여 E2E 테스트 코드 생성.
기존 테스트 패턴을 분석하여 일관된 스타일로 작성.
