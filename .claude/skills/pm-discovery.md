# PM Discovery Chain

Teresa Torres + Pawel Huryn 프레임워크 기반 5단계 Discovery 체인.

## 5단계 프로세스

1. **Brainstorm**: 기회 발굴 (Opportunity Discovery)
2. **Assumptions**: 핵심 가정 추출 및 위험도 평가
3. **Prioritize**: ICE/RICE 스코어링으로 우선순위 결정
4. **Experiments**: 가정 검증 실험 설계 (최소 비용)
5. **OST Synthesis**: Opportunity Solution Tree 종합

## 사용법

PM이 새 기능 기획 시 `/pm-discovery` 호출하여 체계적 Discovery 수행.
결과물은 PRD의 Discovery 섹션으로 통합.
