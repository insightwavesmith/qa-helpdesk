## CLAUDE.md 읽고 delegate 모드로 팀원 만들어서 진행해라

## 배경
어제 Plan은 완료됐고, Design 작성 직후 중단됐음.
마케팅팀 역할 재정의: 소재 분석이 아니라 **자사몰사관학교(bscamp) 강의 자체를 홍보**하는 것.

## 핵심 방향
우리 콘텐츠 파이프라인(정보공유, 뉴스레터 등)의 콘텐츠를 다양한 오가닉 채널에 자동 배포하는 프로세스 기획

## TASK: Design 문서 완성
5개 채널별로 구체적 배포 프로세스 설계:

### 1. 이메일/뉴스레터
- 기존 뉴스레터 시스템 활용 (Solapi/SMTP)
- 발송 주기, 대상 세그먼트, 자동화 트리거

### 2. 블로그 (네이버 blog.naver.com/insight-wave)
- 콘텐츠 변환 포맷 (정보공유 → 블로그 글)
- 자동 포스팅 API 또는 수동 프로세스
- SEO 키워드 전략

### 3. 유튜브
- 콘텐츠 유형 (강의 클립, 성과 사례, 팁)
- 제작 파이프라인
- 업로드 주기

### 4. 인스타그램
- 콘텐츠 유형 (카드뉴스, 릴스, 스토리)
- 제작 템플릿
- 해시태그 전략

### 5. 정보공유 (bscamp 내 콘텐츠)
- 기존 콘텐츠 큐레이션 → 외부 채널 배포 연동
- 자동 변환 규칙
